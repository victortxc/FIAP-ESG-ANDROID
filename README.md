## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/victortxc/FIAP-ESG-ANDROID.git && cd FIAP-ESG-ANDROID
```

Follow the steps below

```bash
# Start the project
$ Run an Android simulator
$ Run the project via Android Studio
```
